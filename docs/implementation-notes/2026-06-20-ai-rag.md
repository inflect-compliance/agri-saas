# 2026-06-20 ÔÇö Agricultural RAG (feat/ai-rag)

**Commit:** `<sha>` feat/ai-rag ÔÇö pgvector + KnowledgeChunk + licensed corpus + hybrid retrieval

## Design

Make the general model agricultural via retrieval (RAG beats fine-tune for
facts): grounded, cited answers from a curated, **licence-gated** GLOBAL corpus
plus the tenant's own data, with strict tenant isolation.

Flow: ingest (licence-checked) ÔåÆ chunk ÔåÆ embed (`AiProvider.embed`, Ollama
`nomic-embed-text` local) ÔåÆ store in `KnowledgeChunk` (pgvector) ÔåÆ `retrieve()`
(hybrid keyword Ôèò vector) ÔåÆ `buildContext()` (numbered citable sources, answer-
only-from-sources) ÔåÆ `askKnowledgeBase()` (`complete()` or a fixed "not in my
sources" refusal).

## Decisions

- **Nullable `tenantId` = GLOBAL catalog vs tenant-private**, guarded by ONE
  asymmetric RLS policy (`USING (tenantId IS NULL OR own) WITH CHECK (own)`),
  not a split policy. A permissive split INSERT policy would implicitly grant
  `WITH CHECK(true)` on UPDATE and let an `app_user` move a NULL (GLOBAL) row to
  any tenant ÔÇö the single-policy form forecloses that. Registered in
  `SINGLE_POLICY_EXCEPTIONS` (it's a deliberate catalog-variant).
- **pgvector mirrors PostGIS** ÔÇö Prisma can't model `vector`, so the column is
  `Unsupported("vector(768)")` and all vector reads/writes go through raw
  `Prisma.sql`/`$executeRaw` in `src/lib/db/embeddings.ts`, exactly like geo.ts.
- **Licence gating is enforced in code, not just docs** ÔÇö `assertLicensedSource`
  refuses anything off the allowlist (KCC/GODL, FAIR-Forward/Digital Green, EU
  2018/848, USDA 7 CFR 205) and hard-refuses GlobalG.A.P. (copyrighted; the
  product cites it, never ingests its text). Every chunk records its `source`.
- **Hybrid retrieval + defence-in-depth** ÔÇö keyword (`contains`) Ôèò vector
  (cosine NN) merged and ranked; RLS isolates tenants, and the app query ALSO
  filters `tenantId` (query-shape + tenant-isolation guards, and clarity).
- **Forced grounding** ÔÇö `buildContext` instructs cite-by-number and a fixed
  refusal string; `askKnowledgeBase` short-circuits to that refusal (no model
  call) when retrieval is empty, so the model can't hallucinate past empty
  sources.
- **Local-first, zero-cost** ÔÇö embeddings run on the same Ollama OpenAI-
  compatible endpoint as the chat model (`nomic-embed-text`); prod swaps
  `AI_EMBED_MODEL`/`AI_BASE_URL` like the rest of the provider.
